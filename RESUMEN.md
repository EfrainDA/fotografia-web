## 🎨 Nueva Paleta de Colores

### Colores Principales
```css
--bg-dark: #0a0e27        /* Fondo oscuro profundo */
--bg-card: #1a1f3a        /* Fondo para tarjetas */
--bg-light: #2d3561       /* Fondo más claro */

--accent: #00f0ff         /* Cyan principal */
--accent-secondary: #ff0068      /* Magenta */
--accent-tertiary: #a855f7       /* Púrpura */
--accent-quaternary: #06b6d4     /* Cyan secundario */

--text: #f0f4ff           /* Texto principal */
--text-secondary: #b0b8d6 /* Texto secundario */
```

### Efectos de Brillo (Glow)
- `--glow-cyan`: Brillo cian sutil
- `--glow-magenta`: Brillo magenta sutil
- `--glow-purple`: Brillo púrpura sutil
